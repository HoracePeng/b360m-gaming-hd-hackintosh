# b360m-gaming-hd-hackintosh
办公用电脑，手动黑了一台用于开发，网上查b360m gaming hd这个板子的efi不多，就发出来给需要的人, `注意要自己替换序列号`


## 我的配置

|         硬件       |                   型号                     |
|-------------------:|:------------------------------------------|
|               主板 | 技嘉 B360M Gaming HD                       |
|             处理器 | 英特尔酷睿 i5-8500                          |
|               显卡 | Intel UHD Graphics 630            |
|        无线 + 蓝牙 | 无                          |
|             硬盘1 | 金士顿 A400系列 SSD 240G  |
|             内存 | 威刚 万紫千红 DDR4 2600 8G * 2  |
